# Car Insurance Claim Prediction System

An end-to-end Machine Learning web app that predicts whether a customer is likely to file an insurance claim.

## Features
- Logistic Regression & Random Forest
- ROC-AUC Comparison
- Confusion Matrix
- ROC Curve
- Risk Gauge Meter
- Streamlit Dashboard

## Built With
Python, Scikit-learn, Streamlit, Plotly
